# Product Planning Pack (P0-01 to P0-10)

This directory contains the first P0 execution artifacts for the Shadow API MVP.

## Deliverables

- `icp-and-verticals.md` (P0-01)
  - Ideal Customer Profile (ICP)
  - Top 3 launch verticals with rationale
- `demand-scorecard.md` (P0-02)
  - Candidate source-site demand and feasibility scoring
- `legal-risk-matrix.md` (P0-03)
  - Platform-level risk matrix and operating guardrails
- `mvp-endpoint-catalog.md` (P0-04)
  - MVP API endpoint set, priorities, and contracts
- `response-schema-conventions.md` (P0-05)
  - Standard response envelope, metadata, and error schema
- `slos-and-reliability-baseline.md` (P0-06)
  - Baseline SLOs and SLI definitions
- `pricing-hypotheses.md` (P0-07)
  - Subscription and pay-per-result pricing hypotheses
- `north-star-metrics-events.md` (P0-08)
  - North-star KPI and analytics event taxonomy
- `demo-use-cases.md` (P0-09)
  - Three end-to-end launch demo use cases
- `prd-v1.md` (P0-10)
  - MVP product requirements document

## Scope Notes

- This pack is for MVP planning and prioritization, not legal advice.
- Any production rollout must include legal review for each platform and region.
